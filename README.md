# Wine Cultivar Origin Prediction System

This project predicts the origin (cultivar) of wine based on chemical properties using a machine learning model.

## Features Used
- Alcohol
- Malic Acid
- Alcalinity of Ash
- Total Phenols
- Flavanoids
- Proline

## Technologies
- Python
- Scikit-learn
- Flask
- HTML/CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000/
